## Start Script

Cloning the Repository
```
git clone https://github.com/Hisoka-Morrou/hisoka-baileys.git
```

Go to the project directory and install all dependencies.</br>
```sh
cd hisoka-baileys

npm i
```

Finally, run the command below to start :
```sh
# non pm2
npm start

# with pm2
npm run dev
```
